CSS display property:- .) Inline:- Take as much width as required. Notable to set height,width,top/bot padding and top/bot margin.place element beside to each other not take a newline. .) Block:- Take full width.Place element vertically not side by side and also we can set padding,margin,width and height. .)Inline block:- Same as inline.But we can set width,height,margin and padding.Element also move next to each other. .)display none:- Remove element from document and it's space is not reserved or blocked. .)Visibility none:- Element is just not visible and it's space is reserved. .)text-align property used in block element to align text horizontally.
